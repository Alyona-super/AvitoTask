## Тест-кейсы на пользовательские сценарии:
# 1.	Создание задачи
Предусловие: Нахождение на главной странице таск-трекера

1.1.	Нажать на кнопку «создать задачу»

1.2.	Заполнить поле «Название»

1.3.	Заполнить поле «Описание» (опционально)

1.4.	Выбрать проект из выпадающего списка

1.5.	Выбрать приоритет из выпадающего списка

1.6.	Выбрать исполнителя из выпадающего списка

Ожидаемый результат:
Задача появляется в списке

# 2.	Открытие карточки задачи
Предусловие: Нахождение на главной странице таск-трекера, наличие с списке задач хотя бы одной задачи

2.1.	Нажать на заголовок задачи

Ожидаемый результат:
Открытие окна с подробной информации задачи

# 3.	Поиск задачи
Предусловие: Нахождение на главной странице таск-трекера, наличие с списке задач хотя бы одной задачи

3.1.	Нажать на поле «Поиск»

3.2.	Ввести название задачи

3.3.	Выбрать статус задачи (опционально)

3.4.	Выбрать доску (опционально)

Ожидаемый результат:
Отображение задачи

# 4.	Переход на доску проекта
Предусловие: Нахождение на главной странице таск-трекера

4.1.	Нажать на кнопку «проекты»

4.2.	Нажать на ссылку «перейти к доске»

Ожидаемый результат: 
Открытие доски проекта
